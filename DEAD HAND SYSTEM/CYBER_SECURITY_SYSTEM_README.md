## 🚀 **Installation & Setup**

### **Prerequisites:**
- **Java 17+** with Maven
- **Python 3.8+** with pip
- **Node.js 16+** with npm
- **MySQL/PostgreSQL** for database

### **1. Java Security System**
```bash
cd "JAVA/CyberSecuritySystem"
mvn clean install
mvn spring-boot:run
```

### **2. Python Attack Detector**
```bash
cd "PYTHON/CyberAttackDetector"
pip install -r requirements.txt
python attack_detector.py
```

### **3. TypeScript Web Security**
```bash
cd "TS/CyberAttackDetector"
npm install
npm run build
npm start
```

### **4. SQL Database**
```sql
-- Import the advanced e-commerce schema
mysql < "DATA BASE/SQL/AdvancedEcommerce/schema.sql"
mysql < "DATA BASE/SQL/AdvancedEcommerce/sample_data.sql"
```
